# Code Explanation

Provide detailed explanation of code structure, logic, and design decisions.

## Usage
- `/explain-code <file>` - Explain specific file
- `/explain-code <function>` - Explain specific function
- `/explain-code` - Explain current context/selection

## Explanation Includes
1. **Purpose**: What the code does and why
2. **Architecture**: How it fits in the larger system
3. **Data Flow**: Input/output and transformations
4. **Dependencies**: External libraries and their usage
5. **Complexity**: Time/space complexity for algorithms

## Output Format
- Use diagrams (ASCII/Mermaid) where helpful
- Include code snippets with annotations
- Provide examples of usage
